What is statistical significance?

  • Statistical significance indicates that the observed effect is unlikely due to random chance
  • Typically measured using p-values (p < 0.05 considered significant)
  • Helps distinguish between real patterns and random noise in data

What is correlation vs causation?

  • Correlation: Two variables change together (positive/negative relationship)
  • Causation: One variable directly causes change in another
  • Correlation does not imply causation - need controlled experiments to establish causality

What is feature engineering?

  • Process of creating new features from existing data to improve model performance
  • Includes: scaling, encoding categorical variables, creating interaction terms
  • Can significantly impact model accuracy and interpretability
