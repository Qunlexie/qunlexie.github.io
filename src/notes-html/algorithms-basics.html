What is Big O notation?

  • Describes the time/space complexity of algorithms
  • Shows how performance scales with input size
  • Common complexities: O(1), O(log n), O(n), O(n¬≤)

What is a hash table?

  • Data structure that maps keys to values
  • Average O(1) lookup, insertion, deletion
  • Uses hash function to compute array index
  • Handles collisions with chaining or open addressing
