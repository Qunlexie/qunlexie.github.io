Research and Projects

Research

Neural Network Model for Critical Coning Rate Prediction

Carried out Sensitivity Analysis to determining the important input variables and the magnitude of their impacts on critical rates.

Developed a feed-forward layered network with 12 input variables and Bayesian regularization to predict critical rates.

Comparison of results from the Neural Network with other correlations for predicting critical rates.

Link to paper

Bayesian Optimization and Time Series Forecasting for Power Outages

Built a Bayesian optimization and hierarchical time series forecasting models to estimate non-weather-related power outages (NWO) with high accuracy, significantly outperforming baseline ARIMA and SARIMA models.

Defined a robust metric for NWO (non-weather outage count index, NWOCI) and leverage advanced pre-processing and forecasting techniques in Kats and Prophet.

Link to paper

Statistical and Quantitative Analysis of Electricity Price Volatility

Built a web scrapping API to scrape data from New England Independent System Operator (ISONE).

Built a Robust Quantile Regression Model to analyse the effects of Hydro on Electricity Price and Volatility

Link to paper

Data Driven Model for Risk Analysis in Complex Interconnected Human-Natural Systems

Proposed a multi-layer dynamic interconnection model for assessing risk in interdependent systems.

Developed a supporting data scientific framework that uses multiresolution data for knowledge discovery across complex systems.

Projects

Deep learning Time Series Forecasting

Developed a time series regression model to predict future horizons. Both linear and non-linear models, including Prophet and Neural Prophet, were explored. Feature engineering and cross-validation were conducted, with the Hyperparameter-tuned Neural Prophet model achieving the best performance.

Link to GitHub repository

Sentiment Analysis using word Embedding

Developed an Embedding model and a bag of word model (using Glove representation) for feature extraction for raw sentiment. Built Machine Learning models to be able to determine whether a review was good or bad from the sentence structure for each review. Compared and analysed performance of the models.

Link to GitHub repository

Ensemble Based Machine Learning Multiclass Classification

Built and trained a classifier to distinguish between the three classes using a hypothetical dataset. Extensive Feature Engineering including categorical feature encoding and intelligent feature creation. Built three variations of the boosted ensemble model: A pre-tuned model was trained with default parameters, A tuned model from extensive hyperparameter search and a model accounting for class imbalance.

Link to GitHub repository

Machine Learning Model for Predicting Wildfire

Built 11 classification models to predict whether or not there is an occurrence of wildfire using Normalized Difference Vegetation Index (NDVI), Land Surface Temperature (LST) and the Thermal Anomalies. Accounted for class biases by building models with and without weight balance. Performed comparative Analysis the performance of different Machine learning models (with careful hyperparameter selection) for wildfire prediction.

Link to GitHub repository

Extractive Text Summarization for UN SDG

Built unsupervised ML model to for extractive summarization of articles that capture the United Nations Sustainable Development Goals. Data collection, cleaning as well as pre-processing operation for articles including tokenization and removing stop words. Used cosine similarity to generate similarity matrix and rank to produce summarized sentences.
